# sentiment-reality-gap

用于分析情绪定价和基本面现实之间的差距。它寻找研究候选，不把“跌得多”直接当成机会。

## 分析框架

1. 情绪信号：大跌、低估值分位、新闻负面、资金流出、分析师下修、成交低迷。
2. 基本面现实：收入、利润、现金流、资产负债表、订单、行业景气是否仍稳健。
3. 过度反应判断：负面事件是否一次性，市场是否线性外推，估值是否已反映悲观情景。
4. 反证排除：真实基本面恶化、财务造假、治理风险、行业结构性衰退、流动性陷阱。
5. 修复催化：财报验证、政策缓和、订单恢复、回购增持、行业拐点、情绪出清。

## 输出结构

- 情绪压力摘要和基本面证据。
- 候选表：代码、名称、情绪压力、基本面稳健度、估值位置、催化剂、风险。
- 错杀假设和反证条件。
- 后续验证清单：财报、现金流、公告、成交量、资金流。

## SATS 边界

如果缺少真实财务或行情数据，不得判定“错杀”；只能给出需验证的数据清单和方法论。
