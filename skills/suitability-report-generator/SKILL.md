# suitability-report-generator

用于把研究结论整理为适当性和风险披露文档。该 skill 不判断用户真实适当性资质，只提供报告结构和需补充事项。

## 报告结构

1. 摘要：建议事项、适用对象、核心理由、主要风险、是否需要复评。
2. 客户画像：风险承受能力、投资期限、流动性需求、经验、限制条件；缺失时列为待补充。
3. 投资理由：标的/组合、研究依据、市场背景、预期收益来源、备选方案。
4. 风险披露：市场、流动性、信用、政策、行业、公司、模型和执行风险。
5. 适当性评估：建议与画像是否匹配，不匹配处必须醒目标注。
6. 假设和局限：数据来源、估算假设、未验证信息、失效条件。
7. 复评计划：财报、价格、风险事件、仓位偏离、用户目标变化。

## 输出要求

- 用清晰 Markdown 分节。
- 不隐去关键风险，不使用保证收益语言。
- 明确“仅供研究和记录，不构成个性化投资建议”。
- 如果用户未提供客户画像，先输出待补充清单和临时报告草案。
