# undervalued-stock-screener

用于识别“便宜但不是价值陷阱”的 A 股候选。必须同时看估值、质量、现金流和低估原因，不能只按低 PE/PB 排名。

## 筛选框架

1. 估值过滤：PE、PB、PS、股息率、EV/EBITDA 或行业分位；优先使用行业内比较。
2. 基本面质量：ROE、毛利率、净利率、资产负债率、经营现金流、扣非利润趋势。
3. 低估原因：周期低谷、短期事件、行业偏见、流动性折价、治理风险、财务质量问题。
4. 反证排除：ST/退市风险、连续亏损、现金流恶化、商誉减值、应收异常、政策或行业结构性下行。
5. 催化剂：业绩修复、估值修复、分红提升、回购增持、行业景气反转。

## 输出结构

- 筛选条件和可用数据说明。
- 候选表：代码、名称、行业、估值分位、质量指标、低估原因、催化剂、风险等级。
- 个股卡片：业务概述、估值对比、基本面指标、核心风险、估值区间。
- 结论用“研究优先级”表达，不输出确定性目标价或买卖指令。

## SATS 边界

RAG 只提供价值筛选方法；真实估值和财务字段必须来自 SATS 数据源。缺少行业中位数时，只能做候选内相对比较。
