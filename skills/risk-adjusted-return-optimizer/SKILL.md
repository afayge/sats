# risk-adjusted-return-optimizer

用于根据资金规模、风险偏好和投资期限设计研究型组合框架。它不直接交易，也不替代适当性评估。

## 构建流程

1. 确认画像：资金规模、风险偏好、投资期限、现金需求、可承受回撤、账户约束。
2. 风险偏好：保守型重视回撤控制，稳健型平衡收益与波动，积极型承受更大权益波动。
3. 资产配置：权益、现金、固收/货币、ETF/指数、行业主题、卫星个股。
4. 核心-卫星：核心仓用宽基/低波/红利/质量，卫星仓承载行业或个股观点。
5. 仓位约束：单票、单行业、主题、流动性和高波动资产设置上限。
6. 再平衡：定期 + 阈值混合策略；考虑 A 股印花税、佣金和分红税。
7. 下行保护：现金缓冲、止损规则、低相关资产、降低高 Beta 暴露。

## 输出结构

- 投资者画像和关键假设。
- 目标配置表：资产类别、权重区间、用途、风险。
- 候选工具或股票池说明。
- 再平衡规则和触发条件。
- 压力情景下的预期表现。
- 不适用情形和风险披露。

## SATS 边界

没有用户画像时只给模板；没有真实持仓或行情时不输出精确组合指标。任何配置建议都必须标注为研究参考。
