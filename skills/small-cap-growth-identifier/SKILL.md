# small-cap-growth-identifier

用于分析被忽视的小市值成长公司。该 skill 强调“成长质量”和“被发现催化”，同时必须把流动性、治理和估值波动放在显眼位置。

## 筛选标准

1. 市值范围：按用户定义或以中小市值候选为主；明确排除壳价值和纯概念炒作。
2. 成长：营收、扣非利润、毛利率、订单或产能扩张是否持续。
3. 资产负债表：现金、短债、应收、存货、资本开支和经营现金流匹配。
4. 管理层：实控人稳定、管理层持股、股权激励、董监高增减持。
5. 机构发现潜力：低覆盖、低持仓、行业催化、专精特新、国产替代或新产品放量。
6. 风险：流动性、解禁、单一客户、利润波动、财务质量和信息披露不足。

## 输出结构

- 候选表：代码、名称、市值、增长、质量、估值、流动性、催化剂。
- 个股卡片：商业模式、核心数据、估值、管理层、机构持仓、增长驱动、执行风险。
- 为什么被忽视，以及可能被市场重新发现的触发条件。

## SATS 边界

没有真实市值、财务或成交额数据时，不得臆造排名；只能输出筛选框架或要求补充数据。
