# technical-basic

用于解释 SATS 指标系统输出。

## 分析框架

- 趋势：MA5/10/20/60/120 与价格位置。
- 动能：MACD 金叉/死叉、柱体变化。
- 超买超卖：RSI 6/12/24 与 KDJ。
- 波动：BOLL 位置和 ATR。
- 结构：支撑/压力、量价配合、蜡烛图确认。

回答应说明指标滞后性，不构成投资建议。
