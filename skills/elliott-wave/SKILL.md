# elliott-wave

SATS 第一版艾略特波浪是启发式辅助信号，不是精确自动数浪。

## 回答原则

- 用“潜在浪型”“可能处于”这类谨慎表述。
- 输出关键峰谷、候选浪段、置信度和失效位。
- 与趋势、MACD、成交量和支撑压力交叉验证。
- 不把波浪分析作为单独交易依据。
