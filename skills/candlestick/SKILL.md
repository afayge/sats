# candlestick

SATS 已在指标系统中支持常见蜡烛图形态。本 skill 用于解释这些形态的含义和局限。

- 单根 K 线必须结合趋势位置和成交量解释。
- 看涨/看跌吞没需要比较前后实体关系。
- 三根形态需要确认前置趋势和第三根确认力度。
- 不要只凭形态给出买卖结论，应结合支撑压力、量能和资金流。
