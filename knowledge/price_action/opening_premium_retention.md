# 开盘溢价率去留判断知识卡

来源：用户提供的股票经验截图整理，主题包括开盘溢价率、涨停后持仓去留、盘中强弱判断、减仓和风险离场。

## 风险声明与使用边界

本文仅作为 SATS 本地知识库中的方法论、交易纪律和风险提示，不构成任何投资建议。

Agent 或 LLM 调用本文时，必须遵守以下边界：

- 本文不能替代真实行情、分钟线、成交量、封单、资金流、市场情绪、回测或交易系统数据。
- 涉及个股去留判断时，必须通过 SATS 的结构化行情数据验证昨日收盘价、今日开盘价、盘中涨跌幅、成交量和关键价位。
- 本文只提供短线持仓复核框架，不直接给出买入、卖出或继续持有建议。
- 如果真实数据与本文经验冲突，以真实数据、风险控制和用户确认优先。

核心原则：手里的股票涨停后，不要只靠感觉猜去留；先看开盘溢价率，再结合盘中走势和成交量确认。

## 开盘溢价率公式

开盘溢价率用于衡量今天开盘价相对昨日收盘价的强弱。

计算公式：

```text
开盘溢价率 = (开盘价 - 昨日收盘价) / 昨日收盘价 * 100%
```

使用前必须确认：

- 昨日收盘价真实可靠。
- 今日开盘价真实可靠。
- 股票是否处于涨停后、强势连板、断板修复或普通波动场景。
- 是否存在停牌、除权除息、复牌、重大公告等会扭曲开盘价的特殊情况。

## 四档去留框架

算出开盘溢价率后，按四档进行纪律复核。

### 1. 溢价率大于 5%

含义：强势抢筹信号。

经验解读：

- 说明资金非常看好。
- 如果盘中不跌破关键弱化位，可以按纪律继续观察。
- 连板概率相对更高，但仍需要盘中确认。

需要继续验证：

- 开盘后是否快速拉升，而不是高开低走。
- 盘中是否跌破 3% 以内的强势防线。
- 成交量是否健康，是否出现异常放量出货。
- 是否有涨停封板、封单强度和炸板回封情况。

风险提示：

- 高溢价不是无风险信号。
- 如果高开后快速放量跳水，可能是资金兑现而非继续抢筹。
- 不能因为开盘溢价高就忽略止盈和止损纪律。

### 2. 溢价率 2% 到 5%

含义：健康走势。

经验解读：

- 说明承接尚可，走势相对健康。
- 如果盘中继续冲高，可按短线纪律分批锁定利润。
- 若冲高到 7% 以上，建议先把利润装进口袋，至少考虑减仓一半。

操作纪律：

- 冲高到 7% 以上，优先分批减仓，避免利润回吐。
- 不要因为走势看起来健康就满仓硬扛。
- 后续是否继续持有，需要看回落承接、量能和关键价位是否守住。

风险提示：

- 2% 到 5% 不是必涨区间，只是说明开盘承接不弱。
- 如果冲高回落且量能异常，应按风险处理。
- 先保护利润，再谈继续持有。

### 3. 溢价率 0% 到 2%

含义：危险信号。

经验解读：

- 开盘溢价偏弱，说明资金态度不够坚决。
- 如果开盘半小时仍然不拉升，不要抱有幻想。
- 需要及时止盈离场或严格降低风险，避免回落。

操作纪律：

- 观察开盘半小时是否有明确拉升。
- 如果半小时仍无力上攻，应按风险处理。
- 不要把弱开盘误认为主力洗盘。

风险提示：

- 弱溢价容易从小赚变成回撤。
- 如果继续走弱，可能说明前一日强势已经衰减。

### 4. 溢价率小于 0%

含义：负溢价，主力出逃风险。

经验解读：

- 低于昨日收盘价开盘，说明开盘承接明显偏弱。
- 对涨停后或强势股来说，负溢价常常是风险信号。
- 不要犹豫，不要幻想，优先离场或严格风控。

操作纪律：

- 开盘直接走弱时，不要盲目等待反抽。
- 若再不走，后续被深套概率增大。
- 负溢价叠加放量下杀时，应优先按主力出逃处理。

风险提示：

- 负溢价不是低吸理由。
- 强势股负溢价往往意味着情绪转弱或资金兑现。

## 盘中确认条件

开盘溢价率只是第一步，不能单独决定去留。还需要结合盘中走势确认。

重点检查：

1. 开盘半小时是否拉升。
2. 是否高开低走。
3. 是否放量下杀。
4. 是否触及涨停或接近涨停。
5. 冲高后是否快速回落。
6. 回落时是否跌破关键支撑或昨日收盘价。
7. 成交量是健康承接，还是异常放量兑现。
8. 大盘、板块和同题材个股是否同步走弱。

## Agent 调用检查清单

当 agent 使用本知识库复核股票去留时，应按以下顺序验证：

1. 补充昨日收盘价。
2. 补充今日开盘价。
3. 计算开盘溢价率。
4. 判断溢价率所属档位：大于 5%、2% 到 5%、0% 到 2%、小于 0%。
5. 补充开盘半小时走势：拉升、横盘、高开低走或低开下杀。
6. 补充当日最高价、最低价、当前价或收盘价。
7. 补充盘中最大涨幅，确认是否冲高到 7% 以上。
8. 补充成交量变化，识别健康承接还是放量出货。
9. 检查是否涨停、炸板、回封或跌破关键价位。
10. 输出符合条件、不符合条件、无法验证条件和风险提示。

Agent 输出时必须同时给出：

- 开盘溢价率计算过程。
- 所属档位和对应纪律。
- 仍需补充的真实数据。
- 不确定性和风险提示。
- “不构成投资建议”的声明。
