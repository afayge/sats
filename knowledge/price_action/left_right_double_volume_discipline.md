# 左倍量与右倍量纪律知识卡

来源：用户提供的股票经验截图整理，主题包括左倍量、右倍量、量能先于价格、主力吸筹、散户追涨杀跌、黄金坑、洗盘、逃顶和风险控制。

Agent 或 LLM 调用本文时，必须遵守以下边界：

- 本文不能替代真实行情、K 线、成交量、分时走势、盘口、资金流、均线、筹码结构、回测或交易系统数据。
- 截图中的收益故事属于个人经验分享，不能外推为可复制收益。
- 本文只提供量价观察和交易纪律检查框架，不直接给出买入、卖出或继续持有建议。
- 涉及个股结论时，必须通过 SATS 的结构化行情数据验证成交量倍数、K 线位置、趋势阶段、关键均线、分时承接和后续确认。

核心原则：新手盯价格，老手看量能。量在价先，量能变化通常早于价格波动。所谓“左倍量抄底，右倍量逃顶”，重点不是机械买卖，而是识别主力资金进出痕迹，并把位置、量能、K 线和后续确认放在一起看。

## 倍量的基础定义

本文中的“倍量”不是普通放量，而是指当日成交量较前一日放大 1 倍及以上的明显异动。

经验含义：

- 这种规模的量能通常不容易由散户零散交易形成。
- 只有主力资金集中进场或离场，才更容易打出明显倍量信号。
- 倍量必须结合位置判断，脱离位置谈倍量，容易误判。

快速区分：

- 上涨阶段看主力参与，重点观察量能是否持续配合。
- 下跌阶段看散户行为，重点观察恐慌、承接和主力是否缺席。
- 低位倍量阳柱偏机会，高位倍量阴柱偏风险。

## 左倍量：主力吸筹的启动暗号

左倍量多出现在股价相对低位、横盘筑底或趋势启动初期，常被视为主力真金白银吸筹的直接痕迹。

主力想在低位拿到足够筹码，通常有两类方式：

- 通过长期阴跌磨底，消耗散户耐心，迫使散户割肉。
- 主动放量承接散户抛盘，快速完成建仓。

所谓左倍量，本质上是在底部区域观察主力是否开始主动吸筹。

左倍量的核心在于位置：

- 股价前期累计跌幅不低于 30% 时，底部信号参考价值更高。
- 股价处于长期均线下方或长期调整后，位置更接近“低位”。
- 低位出现阳柱倍量，通常比高位出现阳柱倍量更值得观察。

### 阳柱左倍量

阳柱左倍量是比较标准的底部启动信号。

候选条件：

- 股价处于相对低位。
- 当日放量收阳。
- 成交量较前一日放大 1 倍及以上。
- 阳线实体不小于 2%。
- 后续没有跌破倍量 K 线低点。

经验解读：

- 说明主力开始主动拿货。
- 后续若出现缩量回踩，且不跌破倍量低点，属于典型的观察区。
- 如果回踩量能缩到倍量当日的 50% 以下，可能形成“黄金坑”。
- 洗盘结束后，如果成交量恢复到倍量水平，股价逐步突破前期震荡平台，才更像主升浪启动。

执行纪律：

- 不能只看到倍量阳线就追。
- 需要等待缩量回踩、倍量低点不破和后续放量突破确认。
- 若倍量后直接冲高回落、后续无承接，不按主力吸筹处理。

### 阴柱左倍量

阴柱左倍量需要重点看出现位置。

低位阴柱左倍量：

- 若出现在长期下跌后的低位，可能是主力通过恐慌盘拿筹。
- 需要观察后续是否快速止跌、缩量回踩和重新放量修复。
- 不能因为阴线倍量就立即恐慌，也不能不等确认就盲目抄底。

高位阴柱左倍量：

- 若出现在高位并伴随冲高回落，往往是主力悄悄出货的前兆。
- 看似放量下跌，实则可能是在暗中派发筹码。
- 后续股价大概率面临较大杀跌风险，这类形态不宜盲目介入。

## 右倍量：逃顶与风险辅助信号

右倍量多出现在股价上涨末期，核心反映的是散户行为，也是主力出货的辅助信号和高位离场参考。

主力在高位出货时，常见手法是：

- 故意拉升股价。
- 制造强势上涨假象。
- 吸引散户跟风追涨。
- 同时悄悄派发筹码。

右倍量的核心价值不是找买点，而是辅助判断风险。

### 阳柱右倍量

阳柱右倍量是典型的散户追涨信号。

候选特征：

- 股价处于上涨末期或相对高位。
- 当日放量收阳。
- 成交量较前一日放大 1 倍及以上。
- 阳线实体逐步缩小。
- 出现放量滞涨，看似强势，实际推动效率下降。

经验解读：

- 可能是散户跟风盘堆积。
- 主力可能趁强势氛围派发筹码。
- 后续容易出现洗盘或回落。
- 盲目追高极容易成为主力收割对象。

执行纪律：

- 高位看到阳柱右倍量，切勿简单追高。
- 需要观察是否冲高回落、分时承接变弱、量能无法继续有效放大。
- 若后续跌破关键支撑或无法延续强势，应优先控制风险。

### 阴柱右倍量

阴柱右倍量常表现为散户恐慌抛售。

候选特征：

- 股价处于相对高位或阶段性下跌中。
- 当日放量收阴。
- 阴线实体较长。
- 恐慌盘集中涌出。

经验解读：

- 看似行情彻底走坏，但恐慌之后有时会出现短期反弹。
- 这类反弹属于短线修复，不等于趋势重新向上。
- 如果反弹量能无法持续放大，需要及时离场。

执行纪律：

- 看到阴柱右倍量，不必立刻恐慌割肉，但也不能幻想长期反转。
- 等待反弹修复后再从容处理，是更稳的风险动作。
- 若反弹失败、关键支撑继续跌破，应果断离场。

## 真倍量与假放量的区分

很多误判来自分不清真倍量与假放量。

真倍量需满足两个核心条件：

1. 同色两根量柱长短差异明显，即使相差约 1/4，也可以视作有效差异。
2. 三根量柱应体现为右倍量的延续形态，不能出现杂乱无章的量能波动，量能整体要有序变化。

标准确认倍量：

- 两柱同色同长。
- 量能翻倍。
- 指导意义最强。
- 实战中较少见。

多数实战场景中，倍量可能呈不规则形态，例如 1/4、3/4 等比例。只要符合“同色差异”的核心条件，并结合位置和走势验证，仍有参考价值。

额外验证：

- 早盘形成的倍量，通常比尾盘放量的可信度更高。
- 倍量有效性要结合分时走势、承接、均线和后续 K 线确认。
- 不能只凭一天的放量做最终判断。

## 常见误区

### 1. 脱离位置谈倍量

左倍量的核心是位置。低位阳柱可能是机会，高位阴柱可能是风险。

如果不看位置，只看倍量，容易把主力出货误判为主力吸筹。

### 2. 把右倍量当成左倍量

多数亏损来自在高位追高接盘，在恐慌时割肉离场。

右倍量更适合用来判断风险，而不是用来确认买点。

### 3. 把单根倍量当成结论

倍量只是候选信号。真正可用的信号，需要后续验证：

- 是否缩量回踩。
- 是否不跌破倍量低点。
- 是否重新放量突破。
- 是否仍处于低位或主升启动初期。

### 4. 忽略趋势和大环境

左倍量抄底也不能逆着大级别下跌趋势硬做。

右倍量逃顶也不能忽略板块、指数和资金流环境。若真实数据表明趋势和承接尚未走坏，应以数据验证为准，而不是机械套用。

## Agent 调用检查清单

当 agent 使用本知识库分析“左倍量/右倍量”时，应按以下顺序验证：

1. 补充真实日线和必要的分时数据。
2. 计算当日成交量是否较前一日放大 1 倍及以上。
3. 判断量柱颜色：阳柱倍量还是阴柱倍量。
4. 判断出现位置：相对低位、横盘筑底、趋势启动初期、上涨末期或高位。
5. 检查前期累计跌幅是否不低于 30%，以及是否位于长期均线下方。
6. 检查阳线实体是否不小于 2%，阴线实体是否明显偏长。
7. 检查是否出现缩量回踩，且是否不跌破倍量 K 线低点。
8. 检查回踩量能是否缩到倍量当日的 50% 以下。
9. 检查后续是否恢复到倍量水平并突破前期震荡平台。
10. 检查是否存在高位放量滞涨、冲高回落或分时承接变弱。
11. 判断倍量是主力吸筹、主力出货、散户追涨还是恐慌抛售的候选信号。
12. 输出符合条件、不符合条件、无法验证条件和风险提示。

Agent 输出时必须同时给出：

- 倍量类型：左倍量、右倍量、阳柱左倍量、阴柱左倍量、阳柱右倍量或阴柱右倍量。
- 成交量倍数和计算依据。
- 出现位置和趋势阶段。
- 后续确认条件：缩量回踩、不破倍量低点、放量突破或反弹失败。
- 风险位和失效条件。
- “不构成投资建议”的声明。
