# 量价关系七种情形知识卡

来源：用户提供的股票经验截图整理，主题包括成交量放大、缩量、横盘、上涨、下跌、量价齐升、量价背离和放量下跌。

## 风险声明与使用边界

本文仅作为 SATS 本地知识库中的方法论、盘面观察框架和风险提示，不构成任何投资建议。

Agent 或 LLM 调用本文时，必须遵守以下边界：

- 本文不能替代真实行情、K 线、成交量、分时走势、盘口、资金流、市场情绪、回测或交易系统数据。
- 涉及个股结论时，必须通过 SATS 的结构化行情数据验证股价方向、成交量方向、相对位置、趋势阶段和关键价位。
- 本文只提供量价关系的经验解释，不直接给出买入、卖出或继续持有建议。
- 如果真实数据与本文经验冲突，以真实数据、风险控制和用户确认优先。

核心原则：量价关系要同时看“价格方向”和“成交量方向”。单看价格或单看成交量，都容易误判。

## 使用前先确认

调用本知识卡前，至少补齐以下数据：

1. 当前或最近一段 K 线的股价方向：上涨、下跌或横盘。
2. 同期成交量方向：放大、缩小、基本不变。
3. 所处阶段：上涨初期、上涨中后期、下跌趋势、相对低位或箱体震荡。
4. 是否突破、跌破、接近涨停、放量冲高或放量下杀。
5. 大盘、板块和同题材个股是否同步走强或走弱。

## 七种量价情形

### 1. 放量横盘

形态：成交量明显放大，但股价没有明显上涨或下跌，整体处于横盘状态。

经验解读：

- 成交量放大通常需要买方和卖方同时活跃。
- 股价横盘说明买卖双方分歧较大，但力量暂时比较平衡。
- 单方面只有买盘或只有卖盘时，成交量往往难以持续放大；有买有卖，才会形成大量成交。

风险提示：

- 放量横盘不是明确方向信号。
- 需要等待后续向上突破或向下跌破确认。
- 若放量横盘后跌破平台，应按风险处理；若放量横盘后突破平台，再结合量能和位置复核。

### 2. 量价齐升

形态：成交量开始慢慢放大，股价也开始慢慢上涨。

经验解读：

- 这是常见的上涨形态，也就是通常说的量价齐升。
- 说明当前处于多方节奏中。
- 股价上涨开始吸引场外跟风资金，资金介入后一起推高股价。

风险提示：

- 量价齐升也需要看所处位置。
- 若已经连续大涨后才出现剧烈放量上涨，可能进入加速或兑现阶段。
- 不能只因为放量上涨就忽略追高、回撤和止盈纪律。

### 3. 价涨量平

形态：股价上涨，但下方成交量没有明显缩量，也没有明显放量，整体接近量平。

经验解读：

- 多空双方处于相对平衡状态。
- 股价可能沿着原来的趋势继续缓慢上行。
- 如果这种情况持续出现，可能说明动能逐渐消耗。
- 另一种解释是主力高度控盘，市场流通筹码较少，只需要少量成交就能推动股价上涨。

风险提示：

- 价涨量平需要结合趋势强度和筹码结构判断。
- 若价格越涨越高但量能长期无法跟上，应警惕动能衰减。
- 若同时存在高控盘迹象，也不能直接等同于无风险上涨。

### 4. 价涨量缩

形态：成交量开始缩小，但股价仍在上涨。

经验解读：

- 这就是常说的量价背离。
- 如果出现在一轮上涨后期，说明已经较难吸引跟风盘，动能开始减弱。
- 如果出现在拉升前期，也可能是主力高度控盘，市场没有多少筹码愿意卖出。

风险提示：

- 价涨量缩不能一概看空，也不能一概看多。
- 上涨后期出现价涨量缩，应重点防范动能减弱和冲高回落。
- 拉升前期出现价涨量缩，需要结合筹码稳定性、盘口承接和关键支撑判断。

### 5. 缩量下跌

形态：成交量开始慢慢变少，股价也跟着选择下跌。

经验解读：

- 说明交易意愿开始逐渐降低。
- 大部分散户可能开始离场，买盘减少，成交量也随之变少。
- 在弱势趋势中，缩量下跌往往说明市场关注度降低，承接不足。

风险提示：

- 缩量下跌不等于跌不动。
- 下跌趋势中的缩量阴跌可能持续很久。
- 需要等待止跌、企稳、支撑和重新放量确认，不能因为缩量就盲目抄底。

### 6. 缩量横盘

形态：成交量开始缩小，股价在区间内小幅横盘。

经验解读：

- 多空双方的交易意愿都不强。
- 市场处于一种平衡或观望状态。
- 方向尚未明确，需要等待放量突破或放量跌破确认。

风险提示：

- 缩量横盘容易演化成突破前整理，也可能演化成下跌中继。
- 不能只凭横盘时间判断方向。
- 需要结合箱体边界、关键均线、支撑阻力和后续量能变化。

### 7. 放量下跌

形态：成交量放大，同时股价下跌。

经验解读：

- 说明空方力量较强。
- 如果股价已经处于下跌趋势中，放量下跌说明抛盘开始加重，卖的人多，买的人少。
- 如果发生在相对低位，可能是恐慌性抛售，也可能是主力刻意打压股价，让散户抛售，主力趁机拿到更便宜的筹码。

风险提示：

- 放量下跌优先按风险信号处理。
- 相对低位的放量下跌不能直接当成买点，必须等待止跌、企稳和承接确认。
- 如果跌破关键支撑且成交量放大，应优先控制风险。

## 快速判断表

| 价格方向 | 成交量方向 | 经验含义 | 默认处理 |
| --- | --- | --- | --- |
| 横盘 | 放大 | 买卖分歧大，双方力量暂时平衡 | 等待突破或跌破确认 |
| 上涨 | 放大 | 量价齐升，多方节奏增强 | 防追高，结合位置确认 |
| 上涨 | 基本不变 | 多空平衡，趋势缓慢延续 | 观察动能是否衰减 |
| 上涨 | 缩小 | 量价背离，或主力高控盘 | 区分上涨阶段，警惕后期转弱 |
| 下跌 | 缩小 | 交易意愿下降，承接不足 | 不因缩量盲目抄底 |
| 横盘 | 缩小 | 交易意愿偏弱，区间平衡 | 等待放量方向选择 |
| 下跌 | 放大 | 空方力量强，抛压加重 | 优先风控，等待止跌确认 |

## Agent 调用检查清单

当 agent 使用本知识库分析量价关系时，应按以下顺序验证：

1. 补充个股或指数的真实 K 线周期：日线、分钟线或周线。
2. 补充股价方向：上涨、下跌或横盘。
3. 补充成交量方向：放量、缩量或量平。
4. 判断所属情形：放量横盘、量价齐升、价涨量平、价涨量缩、缩量下跌、缩量横盘、放量下跌。
5. 补充所处阶段：上涨前期、上涨后期、下跌趋势、相对低位或箱体震荡。
6. 检查是否接近关键支撑、阻力、平台边界、均线或前高前低。
7. 检查后续是否出现突破、跌破、止跌、企稳或回落。
8. 结合大盘、板块、同题材个股和资金流，判断该量价信号是否被外部环境强化或削弱。
9. 输出符合条件、不符合条件、无法验证条件和风险提示。

Agent 输出时必须同时给出：

- 量价情形归类。
- 归类所依赖的真实行情证据。
- 仍需补充的数据。
- 与该情形对应的风险提示。
- “不构成投资建议”的声明。
